Template7 Template React Component

Sometimes you may need to use Template7 templates, like in Virtual List component. For this case there is a special React component.

Usage example

<t7-template>
  <Card title="Title" content={'Hello, {{name}}'" footer="Footer}></Card>
</t7-template>

Renders to:

<script type="text/template7">
  <div class="card">
    <div class="card-header">Title</div>
    <div class="card-content">
      <div class="card-content-inner">Hello, {{name}}</div>
    </div>
    <div class="card-footer">Footer</div>
  </div>
</script>
